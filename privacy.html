Política de Privacidad

Esta aplicación utiliza WhatsApp Cloud API únicamente para permitir a los administradores autorizados consultar y gestionar contratos internos.

Datos que se recopilan

  • Número de teléfono de administradores autorizados
  • Mensajes enviados al bot

Uso de la información

La información se utiliza exclusivamente para mostrar información de contratos y no se comparte con terceros.

Almacenamiento y seguridad

Los datos se almacenan de forma segura en infraestructura privada y solo son accesibles por personal autorizado.
